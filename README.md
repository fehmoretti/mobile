## 🛠️ Como rodar o projeto localmente

### Pré-requisitos

- Node.js
- NPM ou Yarn
- Expo CLI (caso esteja usando Expo)
- Android Studio / Xcode (para emuladores)

### Passos

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/MOBILE.git

# Acesse a pasta
cd MOBILE

# Instale as dependências
npm install
# ou
yarn

# Execute o projeto
npm start
# ou
yarn start
